# Blueprint SDK

![Blueprint](/img/blueprint/logo.svg)

A development environment for TON for writing, testing, and deploying smart contracts.

Blueprint is a development environment designed for creating, testing, and deploying smart contracts on The Open Network (TON).

## Quick start

To create a new project, open your terminal and run the following command:

```bash
npm create ton@latest
```

Follow the on-screen instructions to set up your project.

### Key features

- **Simplified development** – streamlines the process of building, testing, and deploying smart contracts.
- **Seamless deployment** – deploy to the Mainnet or Testnet using your preferred wallet (e.g., Tonkeeper).
- **Fast testing** – run multiple smart contracts in an isolated blockchain environment.

### Tech stack
- **FunC compilation** – uses [func-js](https://github.com/ton-community/func-js) to compile FunC code without requiring a CLI.
- **Smart contract testing** – uses [TON sandbox](https://github.com/ton-org/sandbox) for efficient testing.
- **Deployment tools** – supports TON Connect 2.0-compatible wallets and `ton://` deep links for deployment.

### Requirements
- [Node.js](https://nodejs.org/) – a recent version (e.g., v18). Verify your installation by running `node -v`.
- **IDE support** – use an IDE with TypeScript and FunC support, such as [Visual Studio Code](https://code.visualstudio.com/) with the [FunC plugin](https://marketplace.visualstudio.com/items?itemName=tonwhales.func-vscode).
